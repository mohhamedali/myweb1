PowerBI (Feel free to skip this part. I did it as an example! Focus on the rest of the sections!)

Power BI is a business analytics solution that lets you?visualize?your data and share insights across your organization, or embed them in your app or website. Connect to hundreds of data sources and bring your data to life with live dashboards and reports.

Data wrangling, sometimes referred to as data munging, is the process of transforming and mapping data from one "raw" data form into another format with the intent of making it more appropriate and valuable for a variety of downstream purposes such as analytics.

The list of data sources for Power BI is extensive, grouped into the following:

  1. Files: Data can be imported from Excel (.xlsx, xlxm), Power BI Desktop files (.pbix) and Comma Separated Value (.csv).
  2. Content Packs: It is a collection of related documents or files that are stored as a group. In Power BI, there are two types of content packs, firstly those from services providers like Google Analytics, Marketo or Salesforce and secondly those created and shared by other users in your organization.
  3. Connectors to databases and other datasets such as Azure SQL, Databaseand SQL, Server Analysis Services tabular data, etc.

To do basic calculation and data analysis on data in power pivot, we use Data Analysis Expression (DAX). It is formula language used to compute calculated column and calculated field. DAX works on column values. DAX can not modify or insert data. We can create calculated column and measures with DAX but we can not calculate rows using DAX.

Power query is a ETL Tool used to shape, clean and transform data using intuitive interfaces without having to use coding. It helps the user to: Import Data from wide range of sources from files, databases, big data, social media data, etc; join and append data from multiple data sources. Shape data as per requirement by removing and adding data.

A new programming language is used in power query called M-Code. It is easy to use and similar to other languages. M-code is case sensitive language.

Excel Concepts

Conditional formatting allows you to format cells based on certain criteria, making it easy to visually highlight important information or trends in your data.

Data filtering enables you to focus on specific subsets of data within your spreadsheet, making it easier to analyze and understand large datasets.

Sorting and filtering functionality helps organize and streamline your data by arranging it in a desired order and displaying only relevant information based on specified criteria.

Performing basic mathematical operations such as addition, subtraction, multiplication, and division within Excel allows for quick calculations and analysis of numerical data.

Creating charts in Excel visually represents your data, making it easier to identify trends, patterns, and relationships, and communicate insights effectively.

Excel formulas allow for advanced calculations and data manipulation within spreadsheets, enabling users to perform complex analyses and automate processes.

Access Concepts

Access facilitates seamless import and export of data between various Microsoft Office applications, enhancing interoperability and data integration.

Access is well-suited for individual users who need to create and manage databases without extensive technical knowledge, providing a user-friendly interface and intuitive tools.

Access empowers users to efficiently create, organize, and manage databases, offering features for data entry, storage, retrieval, and manipulation.

Access enables users to publish databases to the web, allowing for easy access and collaboration with others, while maintaining security and data integrity.

Access includes a "Tell Me" feature that acts as an intuitive assistant, helping users quickly locate and execute commands or tasks within the application.

Access provides a report view feature that allows users to create, customize, and generate professional-looking reports from their databases, facilitating data analysis and presentation.

PPT Concepts

Formatting options in PowerPoint enable users to customize the appearance of slides, text, images, and other elements, ensuring visual consistency and enhancing presentation aesthetics.

Presentation playback features in PowerPoint allow users to rehearse, review, and deliver presentations with precision, ensuring smooth transitions and timing.

Backstage view in PowerPoint provides access to file management and other essential functions such as saving, printing, sharing, and exporting presentations, enhancing productivity and workflow efficiency.

SmartArt diagrams in PowerPoint enable users to create visual representations of concepts, processes, and relationships, making complex information easier to understand and communicate.

Transitions and animations in PowerPoint add visual interest and dynamic movement to slides, engaging audiences and emphasizing key points during presentations.

The cropping feature in PowerPoint allows users to trim and adjust the size of images, improving visual composition and focus within slides.

Word Concepts

Styles in Word allow users to apply consistent formatting to text and paragraphs throughout a document, enhancing readability and maintaining a professional appearance.

Templates in Word provide pre-designed layouts and formatting options for documents, saving time and ensuring consistency in document creation.

Contextual tabs in Word dynamically appear and provide relevant commands and tools based on the current selection or context within a document, improving user efficiency and workflow.

Nonprinting characters in Word, such as spaces, tabs, and paragraph marks, are symbols that represent formatting and layout elements but are not visible when printing a document, aiding in document editing and formatting.

Edit mode in Word allows users to modify and revise the content of a document, providing a user-friendly interface for text input and editing.

Tab stops in Word define the position where the cursor moves when the Tab key is pressed, facilitating precise alignment and formatting of text within a document.
